# AI Video Pipeline Demo

A Streamlit app demonstrating an end-to-end automated video production pipeline:

**Script text → ElevenLabs TTS voiceover → HeyGen talking-head avatar → FFmpeg assembly → YouTube-ready MP4 + metadata**

Built as a proposal demo for a YouTube automation workflow project.

---

## What It Does

1. Takes a video script as text input
2. Generates a voiceover MP3 via **ElevenLabs** (choice of 4 voices)
3. Submits an avatar video job to **HeyGen** (async — polls for completion)
4. Generates a YouTube title, description, and tags via **Claude (Haiku)**
5. Merges audio + video using **FFmpeg** into a 1080p MP4
6. Offers a download button for the final video + displays suggested metadata

### Demo Mode (no HeyGen key)

If `HEYGEN_API_KEY` is not set, the app runs in **Demo Mode**:
- ElevenLabs TTS still runs (if key is set)
- FFmpeg assembles the audio over a static branded thumbnail
- Full metadata generation still works
- Great for demonstrating the pipeline structure without a HeyGen subscription

## Setup

### Prerequisites

- Python 3.11+
- ffmpeg installed (`brew install ffmpeg` on macOS)
- API keys (see below)

### Install

```bash
cd output/demos/2026-06-08_ai-video-pipeline
pip install -r requirements.txt
```

### Environment Variables

Copy `.env.example` to `.env` and fill in your keys:

```bash
cp .env.example .env
```

| Variable | Required | Description |
|---|---|---|
| `ELEVENLABS_API_KEY` | Yes (for TTS) | ElevenLabs API key — [get it here](https://elevenlabs.io/app/settings/api-keys) |
| `HEYGEN_API_KEY` | Optional | HeyGen API key — [get it here](https://app.heygen.com/settings?nav=API). Without it, the app runs in Demo Mode. |
| `HEYGEN_AVATAR_ID` | If HeyGen set | Your avatar's ID from the HeyGen dashboard |
| `ANTHROPIC_API_KEY` | Yes (for metadata) | Anthropic Claude API key — [get it here](https://console.anthropic.com/settings/keys) |

### Generate thumbnail (Demo Mode only)

```bash
python make_thumbnail.py
```

This creates `sample_data/thumbnail.jpg` — used as the background image in Demo Mode when HeyGen is not configured.

### Run

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## HeyGen Async Timing

HeyGen video generation is **asynchronous**. After clicking "Generate Video":

1. The job is submitted and you receive a `video_id` immediately
2. A "Check HeyGen Status" button appears — click it after ~2-3 minutes
3. When status is `completed`, the avatar video is downloaded and FFmpeg assembles the final MP4

Typical processing time: **2-3 minutes** for a ~60-second script.

## Architecture

See `workflow.md` for the full Mermaid diagram.

```
Script text
    → ElevenLabs API   → output/voiceover.mp3
    → HeyGen API       → output/avatar_video.mp4  (async, 2-3 min)
    → FFmpeg merge     → output/final_video.mp4
    → Claude Haiku     → title + description + tags (JSON)
```
